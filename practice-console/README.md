# Practice — Console Challenges

One folder per level. Each folder is a self-contained `dotnet` solution with:

- An app project you edit (e.g. `Level01/Program.cs`)
- A test project that verifies your code (e.g. `Level01.Tests/`)

## Workflow

```bash
cd Level01-HelloWorld
dotnet build              # compile
cd Level01
dotnet run                # run your code interactively
cd ..
dotnet test               # run the tests
```

Green tests = level done. Go back to the web app and click **Mark Level Complete**.

## If `dotnet` is not installed

Install the .NET 8 SDK: https://dotnet.microsoft.com/download
